# 📞 CallBridge

CallBridge is a native Android application that monitors incoming phone calls in real time and instantly notifies a connected desktop.

## ✨ Features

- 📞 Real-time incoming call detection
- 🤖 Telegram Bot notifications
- 📡 Local Wi-Fi notifications
- 🔌 WebSocket support
- 📋 Call history
- ⚙️ Configurable settings
- 🚀 Background monitoring
- 🔄 Auto start after boot

## 🛠 Tech Stack

- Kotlin
- Android Studio
- XML
- Material Design 3
- Foreground Service
- BroadcastReceiver
- TelephonyManager
- OkHttp
- Gson
- SharedPreferences
